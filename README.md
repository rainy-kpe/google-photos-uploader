# Google Photos Uploader

Monitors a folder and uploads the images and videos to Google Photos

## Prerequisites

Install Node

```
sudo apt-get install nodejs npm
```

## Build

```
npm run build
```

## Test

```
npm run test
```

## Configure

```
npm run start -- config
```

## Start watching

```
npm run start -- watch --folder .
```

## Get help

```
npm run start -- help
```
