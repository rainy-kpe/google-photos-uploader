# Google Photos Uploader

Monitors a folder and uploads the images and videos to Google Photos

## Prerequisites

Install Node

```
sudo apt-get install nodejs npm
```

## Build

```
yarn build
```

## Test

```
yarn test
```

## Configure

```
yarn start -- config
```

## Start watching

```
yarn start -- watch --folder .
```

## Get help

```
yarn start -- help
```
